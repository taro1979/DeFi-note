# CI検証ルール（作業完了前の必須チェック）

## 必須ルール

### 作業終了時にCIを実行する
作業が完了したら、必ず以下を実行して全てパスすることを確認する:

```bash
pnpm test      # テスト実行
pnpm check     # 型チェック
pnpm build     # ビルド確認
```

### 確認項目
1. **修正点がテストされていること** - 変更に対応するテストが存在し、パスしている
2. **既存テストが壊れていないこと** - 全テストがグリーン
3. **型エラーがないこと** - `pnpm check` がパス
4. **ビルドが通ること** - `pnpm build` が成功

### 禁止事項
- CI未実行のまま「完了しました」と報告しない
- テスト失敗を無視して完了扱いにしない
- 型エラーを放置しない
