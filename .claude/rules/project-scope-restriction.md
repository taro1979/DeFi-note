# プロジェクトスコープ制限

## 絶対ルール
全てのファイル操作は **このリポジトリのプロジェクトルート配下のみ** に限定する。

## 許可される操作
- プロジェクトルート配下の全ファイル操作
- `~/.claude/` 配下（メモリ・設定の読み書き）
- `/tmp/` 配下（一時ファイル）

## 禁止される操作
- 上記以外のディレクトリへのアクセス
- `~/.ssh/`, `~/.env`, `~/.aws/` 等への一切のアクセス
- 他プロジェクトへのアクセス
